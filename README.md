# The Game of Life

https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life

This python script runs the game of life


## Usage

```bash
python main.py <figure>
```

The **figure** argument is optional and can take these values:

- beehive
- pulsar
- penta-decathlon
- glider
- lwss
